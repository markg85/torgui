Welcome!
This is "TorGui", a graphical frontend for my Tor project.

You can simply type the name of a series below and it will search for the latest episodes in that series (it uses the TVMaze API for that).
The result will then be displayed in a nice overview.

Everything you see here is available on github if you want to run your own instance or want to contibute.
